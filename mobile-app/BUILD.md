# Build the Veritag APK

This is a runnable Flutter app over the `veritag_sdk` SDK. It **builds to a
working APK** — confirmed: `app-release.apk` (~48 MB, NFC permission) with
Flutter 3.38.5 + the Android SDK at `~/Android/Sdk`. The `android/` scaffold is
generated by `flutter create` so it matches your Flutter version; `build/` (the
APK output) is git-ignored. Reproduce with the commands below.

## Prerequisites
- Flutter SDK (`flutter --version`) with the Android toolchain
  (`flutter doctor` green for Android). That includes the Android SDK + a JDK.

## Build (3 commands)
```bash
cd consumers/arttrust/mobile-app

# 1. Generate the android/ scaffold for your Flutter version.
#    This adds platform folders; it does NOT overwrite lib/ or pubspec.yaml.
flutter create --platforms=android --project-name arttrust_mobile .

# 2. (NFC) add the permission — see snippet below — then resolve deps.
flutter pub get

# 3. Build the installable APK.
flutter build apk --release
# -> build/app/outputs/flutter-apk/app-release.apk
```

Install on a connected device:
```bash
flutter install                      # or:
adb install build/app/outputs/flutter-apk/app-release.apk
```

## Run it live instead of building
```bash
flutter run                          # hot-reload on a device/emulator
```

## NFC permission (for real tag provisioning)
After `flutter create`, add to `android/app/src/main/AndroidManifest.xml`,
inside `<manifest>` (above `<application>`):
```xml
<uses-permission android:name="android.permission.NFC" />
<uses-feature android:name="android.hardware.nfc" android:required="false" />
```
The **SDK self-test** screen works without NFC or a backend (it creates an
on-device identity and signs the binding). The **Verify** screen needs a running
AttestCore (`make run-api`, or the repo-root `docker-compose.yml`); real tag
*minting* additionally needs an NFC chip.

## What you'll see
- **SDK self-test (offline)** → tap "Create identity & sign": the app creates a
  P-256 identity in the platform secure store and signs the Veritag binding,
  showing the public key, binding and signature. This is the SDK running on the
  device, end to end, with nothing else required.
- **Verify a scan** → point it at an AttestCore URL and paste a tag's `u`/`c`/`m`
  to get the verdict (authentic / counterfeit / replayed / unverified_artist).
